# CountryInfo
A java project that consumes a soap api
The soap api is public on this website https://documenter.getpostman.com/view/8854915/Szf26WHn#8d815ca8-9e0e-427f-9b2a-b15264fc5109
This is a java console application that consumes an api with URL="http://webservices.oorsprong.org/websamples.countryinfo/CountryInfoService.wso",
and displays the countries and their code, together with a currency and it's code
